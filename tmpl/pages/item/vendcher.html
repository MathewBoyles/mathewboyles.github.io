Vendcher

Vendcher was created to simplify vouchers. It allowed for users to add, redeem, and add funds to vouchers. It was designed to work with many POS systems.

Vendcher was created for Mybelle and is not available to download or use at this time.

Features

Add & Spend Vouchers

The Vendcher system allowed for users to create vouchers and for them to redeem credit from them at check-out.

Categories

Categories could be used on vouchers so they may be easily managed. Some uses of this feature included vouchers issued for loyalty and vouchers issued as a refund. Categorization also allowed a view of which type of voucher was being used when at check-out.

Activity Log

An activity log was kept so expenditures and other actions could be viewed for each voucher.

Users & Permissions
Users & Permissions

Multiple user accounts could be created - each with specific permissions. This was also reflected in the activity log, which allowed an administrator to see which user/staff member was redeeming vouchers.

Import & Export

Don't lose a thing. The import and export feature allowed for back-ups to be generated.

Dashboard

The dashboard gives an overview of voucher performance, including how much was spent and when. This information could also be easily printed.
