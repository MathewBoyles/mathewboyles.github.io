Proximity Player

Proximity Player is an un-officlal website for Proximity. It was created as fun project, not on commission. The website uses the YouTube API to gather videos uploaded by Proximity and their metadata to categorise them. Video data is saved in the visitor's cache to minimise API demand.

Features

YouTube API

YouTube's API system is used so a list of Proximity's uploads may be fetched.

Cache

Video data is saved to the user's cache to minimize API demand. Changes are questioned automatically to ensure cache is up-to-date.

Categories
Categories

After an API call is made successfully, the videos returned are categorised based on their title and other metadata.
